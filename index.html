High Performance Computing

- Visual Cryptography using GPU and Multi-core

The project’s goal was to implement Visual Cryptography which is a cryptographic technique that encrypts images and converts them to meaningless data. The base technique used was 4 Subpixel VC where every pixel of the input image is expanded into 4 more images. The project is implemented for various image sizes and both on GPU and Multicore. For smaller image sizes Multi-core worked better than GPU and for larger images the GPU worked better than Multi-core.


Distributed Computing

- Implementation of Chandy Lamport Protocol

This project was implemented as part of Advanced operating systems course. The objective of this protocol is to take consistent global snapshots. A process initiating the protocol broadcasts 'snapshot message' to all the other processes and has to get a reply with the time stamp from every process. This time stamp is part of the global snapshot. The process also checks if the communication channels are empty before committing the global snapshot. At a time multiple snaphot protocols can be operational. The processes then write the global snapshots to an output file. The protocol is implemented using Java and Socket Programming.


Distributed Computing

- Implementation of Lamport Mutex Algorithm

This project was implemented as part of Advanced operating systems course. The main aim of this algorithm is to ensure Mutual Exclusion between multiple processes running at the same time. Each node can request access to critical section at any time but the order in which the requests are satisfied depends on the time stamp of the request. Every node maintains an arraylist of critical sections of other nodes and it also maintains another arraylist of pending replies from other processes. A node/process can enter the critical section only if the pending replies arraylist is empty. The algorithm is implemented using Java and Socket Programming.
